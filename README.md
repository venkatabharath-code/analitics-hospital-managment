# Analytics Hospital Management System

Analytics Hospital Management System is a modern React-based healthcare analytics dashboard developed to provide hospital administrators with a centralized platform for monitoring operational performance, hospital statistics, audit activities, and system settings. The application offers an intuitive and responsive user interface that helps users visualize key healthcare metrics, track administrative activities, and manage system configurations efficiently. Built using React, React Router, Recharts, and modern frontend development practices, the project follows a component-based architecture that ensures maintainability, scalability, and a smooth user experience across different devices.

The system includes core modules such as Analytics Dashboard, Audit Logs, and System Settings. Interactive charts and data visualization components are used to present hospital performance insights in a clear and meaningful way. The project demonstrates frontend development best practices, reusable component design, responsive layouts, routing, state management, and healthcare analytics visualization within a hospital management environment.
